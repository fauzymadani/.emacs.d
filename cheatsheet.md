# Emacs Cheatsheet

## Terminology
- `C-x` = hold Ctrl + x
- `M-x` = hold Alt + x
- Buffer = open file (or any pane content)
- Window = split pane inside Emacs
- Frame = the actual OS window

## Survival
| Key | Action |
|-----|--------|
| `C-g` | Cancel anything |
| `C-x C-s` | Save |
| `C-x C-f` | Open file |
| `C-x C-c` | Quit Emacs |

## Navigation
| Key | Action |
|-----|--------|
| `C-f / C-b` | Forward / back one char |
| `C-n / C-p` | Next / previous line |
| `C-a / C-e` | Start / end of line |
| `M-f / M-b` | Forward / back one word |
| `C-v / M-v` | Page down / up |
| `M-< / M->` | Top / bottom of file |

## Editing
| Key | Action |
|-----|--------|
| `C-space` | Start selection |
| `M-w` | Copy |
| `C-w` | Cut |
| `C-y` | Paste |
| `C-/` | Undo |
| `C-s` | Search forward |

## Windows (panes)
| Key | Action |
|-----|--------|
| `C-x 2` | Split horizontal |
| `C-x 3` | Split vertical |
| `C-x o` | Switch pane |
| `C-x 0` | Close current pane |
| `C-x 1` | Close all other panes |

## Buffers
| Key | Action |
|-----|--------|
| `C-x b` | Switch buffer |
| `C-x k` | Kill (close) buffer |
| `C-x C-b` | List all buffers |
